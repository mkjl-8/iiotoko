Profile

初めまして。かじり（@kajiri_design）と申します。

高知大学に進学。就職活動を行う際に、「このままなんとなく決めた会社に入って大丈夫なのか？」と、将来に不安を覚えたため休学をしました。そんな時、web制作を学ぶ機会があり、それからは、のめり込むように勉強しました。その甲斐あってか、短期間でwebサイトを制作する実力をつけることができました。

お客様と十分なヒアリングをした上でデザイン、コーディングさせていただきます。モダンで洗練されたデザインが得意で、新感覚なwebページを作ることが出来ます。

もちろん、ヒアリングだけでも全然大丈夫なので、気軽にご連絡ください。

My-skills

HTML

webページを見ると、大まかな構成が分かります。また、綺麗なコードを心がけているので、納品後もメンテナンスしやすいと思います。

CSS

HTMLに装飾を与えることができます。レスポンシブ対応もできるので、PC用、スマートフォン用のデザインを作ることが出来ます。

Javascript

webページに動きをつけることが出来ます。文字をふわっと浮かばせることが出来たり、写真のスライドを実装することが出来ます。

Ruby on Rails

簡単なwebサービスを作ることが出来ます。最近は、フリマアプリ「メルカリ」と同じサービスを仲間と一緒に作りました。

HTML Template

HTMLテンプレートを使うことで、効率よくwebページが作れます。850個ほどのテンプレートを持っているので、要望に合わせて選ばせていただきます。

Design

デザインをすることがとても大好きで、「いいデザインとは何か」を常に考えています。webページやロゴ、名刺などをデザインすることが出来ます。
